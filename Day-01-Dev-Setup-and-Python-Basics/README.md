## 🧠 Key Concepts

| Concept                 | What It Does                     |
| ----------------------- | -------------------------------- |
| `print()`               | Displays output to the screen    |
| `input()`               | Gets text typed by the user      |
| `f-string`              | Embeds variables inside strings  |
| `datetime.date.today()` | Gets today's date                |
| `random.choice()`       | Picks a random item from a list  |
| `def function():`       | Defines a reusable block of code |
