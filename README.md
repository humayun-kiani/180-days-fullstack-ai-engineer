## 🚀 How to Use This Repository

### For Learners Following Along

```bash
# Clone the entire repository
git clone https://github.com/yourusername/180-days-fullstack-ai-engineer.git

# Navigate to any day
cd 180-days-fullstack-ai-engineer/Day-01-Dev-Setup-and-Python-Basics

# Read that day's README for instructions
# Then run the project
python main.py
```

### For Each Day's Project

Every day folder contains:

- `main.py` — the day's project code
- `README.md` — explanation of concepts and how to run
